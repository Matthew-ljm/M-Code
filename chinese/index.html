是一种由原始分推导出来的分数，它用来反应学生的成绩在全体考生中的相对位置。

是一种体现相对位置的指标，将全体考生的成绩从高到低排序，低于某分数的人数与总人数的百分比，即为该分数的百分等级。

属于标准分的一类，Z分数 = ( 某分值 – 平均分 ) / 标准差。

是指将学生的原始得分转换为A、B、C、D……等级的指标。

即超过（整体）平均分的比率，是一个反应某分值超出（或低于）整体水平程度的数值，即可以用于当次考试间的比较，也可以用于历次考试间的比较。
超均率 = ( 某分值 – 全体平均分 ) / 全体平均分 * 100%

即与（整体）平均分的比率，是一个反应某分值超出（或低于）整体水平程度的数值，即可以用于当次考试间的比较，也可以用于历次考试间的比较。
比均率 = 某分值 / 全体平均分 * 100%
